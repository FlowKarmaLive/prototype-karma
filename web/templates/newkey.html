FlowKarma.Live

This is a one-time "self-destructing" web page that will allow you to download a client cert to install in your browser so that you can use the FlowKarma.Live network.

Do not reload this page, you will get a 404 "Not Found" response. You and only you should know the passphrase below and use the downloaded cert file.

If anything goes wrong you'll need to get a new "join" URL. Good luck!

Passphrase: 
%(pw)s
Write this down. You will need it each time you install the cert file in a web browser. This passphrase CANNOT be recovered, we do not have it and cannot recreate or discover it. If you lose it you will not be able to install the cert file in browsers, but already-installed certs will continue to function.
